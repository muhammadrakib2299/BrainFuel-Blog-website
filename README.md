##  Features
- Blog listing & detailed view with images and author info  
- Bookmarking system (powered by LocalStorage)  
- Light/Dark mode toggle  
- Fully responsive with polished UI  

##  Tech Stack
- React.js  
- Tailwind CSS + daisyUI  
- React Router  
- LocalStorage
